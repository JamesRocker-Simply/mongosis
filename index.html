Mongosis



Custom components for integrating MongoDB and Microsoft SQL Server Integration Services (SSIS)

Mongosis provides a means to read data from MongoDB within SSIS via custom data source and connection manager components. Available collection fields are automatically detected and added as output fields ready to be piped in to any of the standard SSIS destination components.
